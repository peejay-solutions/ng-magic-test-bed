spyOnFunctionsOf
spyOnFunctionsOf(target: any, source?: any)
Parameters :
Name Type Optional Description
target any No

target's methods will be overwritten with spies. Inherited methods will be overwritten at target object and not on the prototype. This avoids unpleasant side effects.
source any Yes

this optional paramter can be used to create additional spies at the target object. For each method on source and its prototypes a spy will be created on the target.
